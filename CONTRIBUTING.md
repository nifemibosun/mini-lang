# Contribution Guide

We welcome contributions to Mini-lang! Here's how you can get involved:

1. **Fork the Repository**: Click the "Fork" button on the GitHub page to create your copy.
2. **Clone Your Fork**:
   ```bash
   git clone https://github.com/your-username/mini-lang.git
   cd mini-lang

3. **Create a New Branch**
   Run the following command to create a new branch:
   ```bash
   git checkout -b feature-name


4. **Make Changes**
   Add new features or fix bugs in the code. Ensure your changes align with Rust's best practices to maintain code quality and consistency.

5. **Test Your Changes**
   Run the following command to ensure that your changes pass all tests and work as expected:

   ```bash
   cargo test

6. **Commit and Push**
   After making your changes, stage them, commit with a clear message, and push your branch to your repository:

   ```bash
   git add .
   git commit -m "Description of changes"
   git push origin feature-name

7. **Create a Pull Request**
   Go to the original repository on GitHub and submit a pull request (PR) for your branch. Include a detailed description of your changes to help the reviewers understand what you've done and why.
